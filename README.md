# BitPhase

BitPhase is a hybrid phaser, tremolo, and digital destruction processor for the Music Thing Modular Workshop Computer.

Built around a four-stage all-pass phaser engine, BitPhase combines classic sweeping phase cancellation with synchronized modulation, tremolo animation, and controllable digital corruption.

The card moves from subtle vintage-style phasing to aggressive glitch processing while maintaining a simple performance-oriented control set.

---

# Features

* 4-stage all-pass phaser engine
* Feedback-enhanced resonance
* Independent phaser and tremolo LFOs
* External clock synchronization
* Dedicated phaser depth control
* Dedicated tremolo depth control
* Digital sample-rate reduction
* Bit-depth reduction
* Random sample-repeat corruption
* Three performance modes
* CV outputs for both modulation sources
* Pulse outputs for synchronization and timing

---

# Modes

## ▲ Up — Phaser

Classic phaser operation.

Features:

* 4-stage all-pass phaser
* Feedback resonance
* Deep moving notches
* Clock-syncable LFO
* Cleanest signal path

This mode focuses entirely on phase-shifting movement.

---

## ● Middle — Mix

Combines the phaser engine with tremolo modulation.

Features:

* Full phaser processing
* Tremolo applied after the phaser
* Independent tremolo depth control
* Smooth animated movement

The result ranges from gentle modulation to deep pulsing textures.

---

## ▼ Down — Burst

Performance-oriented destruction mode.

Changes include:

* Faster modulation movement
* Increased feedback energy
* Sample-rate reduction
* Bit-depth reduction
* Random sample-repeat glitches
* Aggressive digital instability

Burst mode is intended as a creative effect rather than a traditional modulation processor.

---

# Controls

## Main Knob — Rate

Controls both modulation engines.

### Phaser LFO

Range:

* 0.05 Hz to 6 Hz

Uses a squared response curve for precise control at slow speeds.

### Tremolo LFO

Range:

* 0.5 Hz to 20 Hz

Tracks the same Rate control but operates independently from the phaser.

---

## X Knob — Phaser Depth

Controls phaser sweep intensity.

Increasing X produces:

* Wider notch movement
* Stronger phase cancellation
* More pronounced "whoosh" character

At minimum settings the phaser becomes subtle.
At maximum settings the sweep becomes deep and dramatic.
